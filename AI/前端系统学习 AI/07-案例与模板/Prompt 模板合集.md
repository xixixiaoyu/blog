# 写作提示 —— Prompt 模板合集

使用说明：
- 整理一套可复用的 Prompt 模板，覆盖常见任务并附边界与评估建议。

生成目标：
- 模板类型：总结、改写、分类、抽取、指令执行、结构化输出、函数调用、对话控制、安全提示。
- 每个模板包含：使用场景、输入格式、输出格式、示例与失败案例对比、评估 Rubric。
- 统一命名与版本管理，便于在项目内维护与复用。

结构建议：
1. 使用说明与命名约定（字段、占位符、版本）
2. 模板清单（按任务分类）
3. 评估与回归（Rubric、自动化检测）
4. 风险与限制（幻觉、越权、内容审查）
5. 维护与协作（变更记录、审核流程）

输出格式要求：
- Markdown；每个模板一个小节，附输入/输出示例与失败对比。
- 提供 JSON Schema 或结构化输出的校验建议。

质量检查清单：
- 模板可直接使用或少量改动即可使用。
- 有评估与回归机制，避免质量波动。
- 明确风险与边界，提供防御策略。
