# 写作提示 —— WebGPU 与 WebGL

使用说明：
- 面向前端工程师介绍 WebGPU 与 WebGL 在浏览器端推理与加速中的作用。

生成目标：
- 解释 WebGPU 与 WebGL 的差异、能力边界与适用场景。
- 给出使用框架（如 ONNX Runtime Web、Transformers.js、TensorFlow.js）在 WebGPU/WebGL 上运行的最小示例。
- 提供性能与兼容性注意事项（设备支持、降级策略）。

大纲建议：
1. 能力对比（计算、内存、并行与可编程特性）
2. 生态与支持情况（浏览器/设备兼容性）
3. 基于框架的最小示例（加载模型、执行推理、展示结果）
4. 性能优化（批量、缓存、图编译、数据传输）
5. 兼容与降级（不支持 WebGPU 的替代方案）
6. 安全考量（资源使用、权限、稳定性）

输出格式要求：
- Markdown；代码示例需标注依赖与版本。
- 给出兼容性表或说明链接。

质量检查清单：
- 示例能在现代浏览器中运行或易改造成运行版本。
- 明确降级策略与用户体验保障。
- 性能与资源使用建议具体有效。
