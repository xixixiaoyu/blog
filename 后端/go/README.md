# Go 学习路线与目录

本目录为系统学习 Go 语言的写作与实践模板，包含：语言基础、标准库、并发、Web 与网络、数据库、测试、部署与运维、监控与安全等。

## 推荐学习顺序
1. 语言基础 → 包与模块 → 错误处理 → 并发基础
2. 标准库常用模块（fmt、strings、time、net/http、encoding/json、io、context、sync）
3. Web 开发与网络通信（Gin/Fiber/Echo、HTTP、WebSocket、gRPC）
4. 数据库与存储（database/sql、GORM、Redis 等）
5. 测试与性能（单元测试、基准测试、pprof）
6. 部署与运维（编译、Docker、CI/CD）
7. 监控与追踪、认证与授权、通用工程能力

## 如何使用
- 每个主题下的 Markdown 文件已包含占位清单，按清单逐步补充内容与代码示例。
- 可在「模板」目录中复制文章模板进行写作；示例代码模板可用于快速起步。
- 「示例代码」中提供可运行的 hello 与 http-server 两个示例，便于自测。
