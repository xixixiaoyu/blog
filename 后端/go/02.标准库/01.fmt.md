# `fmt` 包：格式化输入输出的瑞士军刀

请你扮演一位 Go 语言的布道者和资深开发者，为一篇名为《`fmt` 包：格式化输入输出的瑞士军刀》的教程生成内容。

**文章目标：** 帮助 Go 初学者和中级开发者全面掌握 `fmt` 包的各种功能，不仅会用 `Println`，更能熟练运用各种格式化占位符、输入函数以及与 `fmt` 相关的接口，写出更规范、更易于调试的代码。

**内容要求：**

1.  **引言：不仅仅是 `Println`**
    *   从每个 Go 学习者的第一个函数 `fmt.Println("Hello, World!")` 开始。
    *   指出 `fmt` 包的强大之处远不止于此，它是 Go 中处理格式化 I/O 的核心工具，涵盖打印、字符串格式化、文件输出和命令行输入等多种场景。
    *   简要介绍文章将涵盖的主要内容：`Print` 系列、`Fprint` 系列、`Sprint` 系列、格式化占位符、`Scan` 系列以及 `Stringer` 接口。

2.  **三大核心函数家族**
    *   **`Print` 家族：向标准输出打印**
        *   `Println`: 打印并自动换行，参数间自动加空格。
        *   `Print`: 打印，但不自动换行，参数间不加空格。
        *   `Printf`: 核心所在，根据格式化字符串进行打印。这是本节重点。
    *   **`Sprint` 家族：格式化为字符串**
        *   `Sprintln`, `Sprint`, `Sprintf`: 与 `Print` 家族一一对应，但它们不是打印到控制台，而是返回一个格式化好的字符串。这在构建日志、错误信息时非常有用。
    *   **`Fprint` 家族：向 `io.Writer` 打印**
        *   `Fprintln`, `Fprint`, `Fprintf`: 与 `Print` 家族也一一对应，但它们接受一个 `io.Writer` 作为第一个参数，可以将内容写入任何实现了该接口的目标，如文件（`os.File`）、网络连接、或者内存中的 `bytes.Buffer`。这是实现灵活输出的关键。

3.  **`Printf` 的精髓：格式化占位符**
    *   将占位符进行分类讲解，并为每一类提供清晰的示例。
    *   **通用占位符**：
        *   `%v`: 默认格式的值。对于结构体，会打印字段名和值。
        *   `%+v`: 在 `%v` 基础上，打印结构体时会带上字段名。
        *   `%#v`: Go 语法表示的值，会打印出类型的完整声明。
        *   `%T`: 值的类型。
        *   `%%`: 百分号自身。
    *   **布尔值**：
        *   `%t`: `true` 或 `false`。
    *   **整数**：
        *   `%d`: 十进制。
        *   `%b`: 二进制。
        *   `%o`: 八进制。
        *   `%x`, `%X`: 十六进制（小写/大写字母）。
    *   **浮点数**：
        *   `%f`, `%.2f`: 标准表示法，可控制精度。
        *   `%e`, `%E`: 科学计数法。
    *   **字符串与字节切片**：
        *   `%s`: 字符串或字节切片的原生内容。
        *   `%q`: 为字符串加上双引号，并对特殊字符进行转义。
    *   **指针**：
        *   `%p`: 指针的十六进制地址表示。

4.  **自定义类型的格式化：`Stringer` 接口**
    *   介绍 `fmt` 包如何与自定义类型协作。
    *   **`Stringer` 接口**：
        *   定义：`type Stringer interface { String() string }`。
        *   解释：当一个类型实现了 `Stringer` 接口，`fmt` 包（如 `Println`, `%v`）会调用它的 `String()` 方法来获取该类型的字符串表示。
        *   提供一个自定义结构体（例如 `Person`）实现 `Stringer` 接口的例子，展示格式化输出如何变得更友好、更具可读性。

5.  **读取输入：`Scan` 家族**
    *   简要介绍 `fmt` 包也支持从标准输入或 `io.Reader` 读取格式化数据。
    *   `Scan`: 从标准输入读取，直到遇到换行符，将空格分隔的值存入参数。
    *   `Scanln`: 类似于 `Scan`，但在遇到换行符时停止扫描。
    *   `Scanf`: 根据格式化字符串从标准输入读取。

6.  **总结与最佳实践**
    *   回顾 `fmt` 包的核心功能。
    *   提供一些使用建议：
        *   在开发和调试时，多使用 `%+v` 和 `%#v` 来检查变量的详细状态。
        *   为你的核心数据结构实现 `Stringer` 接口，以提高日志和错误信息的可读性。
        *   优先使用 `Sprintf` 构建复杂的字符串，而不是手动拼接。

**代码示例要求：**
*   为每个函数和占位符提供简洁、清晰、可运行的代码示例。
*   在代码注释中解释输出结果，帮助读者理解。
