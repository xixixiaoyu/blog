# AI Prompt: Go "Hello, World!" 示例解析

请你为这个经典的 "Hello, World!" Go 语言示例程序撰写一篇简明扼要的 README 文档。

这篇文档的目标是帮助零基础的 Go 新手理解一个最基本的 Go 程序是如何构成和运行的。

**文档核心内容应包括：**

1.  **示例概述**
    *   说明这是一个标准的 "Hello, World!" 程序，是学习 Go 语言的第一步。
    *   目标：在终端（Terminal）打印出 "Hello, World!" 这句话。

2.  **文件结构**
    *   `main.go`: 存放我们程序代码的源文件。
    *   `go.mod`: Go 模块文件，用于管理项目的依赖。在这个简单的例子中，它主要用于标记项目的根目录。

3.  **代码剖析 (`main.go`)**
    *   逐行解释代码的含义：
        *   `package main`: 声明这个文件属于 `main` 包。在 Go 中，一个可执行程序必须有一个 `main` 包，并且在这个包里有一个 `main` 函数。
        *   `import "fmt"`: 导入 Go 的标准库 `fmt` (format)。这个包提供了格式化输入输出的功能，例如打印文本到控制台。
        *   `func main()`: 定义 `main` 函数。这是整个程序的入口点，当你运行这个程序时，`main` 函数里的代码会被首先执行。
        *   `fmt.Println("Hello, World!")`: 调用 `fmt` 包里的 `Println` 函数，将字符串 "Hello, World!" 打印到标准输出，并在末尾自动添加一个换行符。

4.  **如何运行**
    *   提供清晰的命令行指令：
        ```bash
        # 确保你已经安装了 Go 环境
        # 在当前目录下（hello/）打开终端，然后运行：
        go run main.go
        ```
    *   解释 `go run` 命令的作用：它会编译并直接运行指定的 Go 源文件。

5.  **核心概念总结**
    *   **包 (Package)**: Go 代码的基本组织单位。`main` 包是特殊的，用于创建可执行文件。
    *   **导入 (Import)**: 如何引入外部代码库（标准库或第三方库）。
    *   **函数 (Function)**: `main` 函数是所有可执行程序的起点。

**总结要求：**

请确保语言通俗易懂，对新手友好。避免使用过于复杂的术语，重点是建立起对 Go 程序基本结构的直观认识。
