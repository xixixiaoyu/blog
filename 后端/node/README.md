# Node.js 学习与实践

目标
- 系统化梳理 Node.js 在后端开发中的核心知识、常用库与工程实践
- 形成可迭代的笔记结构，便于后续持续补充与查阅

适用读者
- 具备基本 JavaScript/TypeScript 语法，希望将 Node.js 用于后端服务开发

写作约定
- 示例优先使用 TypeScript，必要时附 JS 版本
- 每个目录以 README.md 作为索引，子主题按一个文件一个主题组织
- 代码块尽量可运行，提供依赖与启动说明

学习路线（建议）
1) 核心基础 → 标准库/模块/事件循环
2) 网络与通信 → HTTP/HTTPS、TCP、WebSockets、GraphQL
3) Web 框架与接口设计 → 中间件、路由、API 版本化
4) 数据库与存储 → SQL/NoSQL、缓存、文件与对象存储
5) 认证与授权、安全 → JWT、OAuth2、CORS、CSRF、输入校验
6) 日志、监控与追踪 → Pino/Winston、OpenTelemetry、Metrics
7) 测试 → 单元、集成、端到端
8) 部署与运维 → PM2、Docker、K8s、CI/CD
9) 微服务与消息、队列与异步任务 → Kafka、RabbitMQ、BullMQ
10) 性能与优化、通用能力、国际化、文档与工具 → 持续完善
