# Python 数据类型

## 最佳提示词

```
请详细介绍 Python 的数据类型系统，包括以下内容：

1. 基本数据类型：
   - 整数（int）：不同进制的表示方法、整数运算
   - 浮点数（float）：精度问题、科学计数法
   - 布尔值（bool）：逻辑运算、真值测试
   - 字符串（str）：创建方式、转义字符、原始字符串

2. 复合数据类型：
   - 列表（list）：创建、索引、切片、常用方法
   - 元组（tuple）：不可变特性、使用场景
   - 字典（dict）：键值对、常用操作
   - 集合（set）：去重、集合运算

3. 类型转换：
   - 隐式类型转换
   - 显式类型转换（int()、float()、str()等）
   - 类型转换的注意事项

4. 类型检查与判断：
   - type() 函数
   - isinstance() 函数
   - isinstance() 与 type() 的区别

5. None 类型：
   - None 的含义和使用场景
   - None 与其他类型的比较

6. 数据类型的选择原则：
   - 何时使用哪种数据类型
   - 内存和性能考虑

请为每种数据类型提供详细的代码示例，包括创建、操作和常见使用场景。解释数据类型的内部实现原理和特点。
```

## 学习要点

- 掌握 Python 的基本数据类型及其特点
- 理解每种数据类型的适用场景
- 学会不同数据类型之间的转换方法
- 掌握复合数据类型的常用操作和方法
- 理解可变与不可变数据类型的区别
- 学会使用类型检查函数验证数据类型
- 了解数据类型的内存使用和性能特点

## 实践练习

1. 创建不同类型的变量并打印它们的类型
2. 练习字符串的各种操作（切片、连接、格式化等）
3. 实现列表的增删改查操作
4. 比较元组和列表的异同，理解不可变性的概念
5. 使用字典存储和检索信息
6. 使用集合进行数据去重和集合运算
7. 练习各种类型转换操作
8. 编写函数判断输入数据的类型

## 代码示例模板

```python
# 基本数据类型示例
integer_var = 42          # 整数
float_var = 3.14159       # 浮点数
bool_var = True           # 布尔值
string_var = "Hello Python"  # 字符串

# 复合数据类型示例
list_var = [1, 2, 3, 4, 5]           # 列表
tuple_var = (1, 2, 3, 4, 5)          # 元组
dict_var = {"name": "Alice", "age": 25}  # 字典
set_var = {1, 2, 3, 4, 5}             # 集合

# 类型检查
print(type(integer_var))              # <class 'int'>
print(isinstance(string_var, str))    # True

# 类型转换
str_num = "123"
int_num = int(str_num)               # 字符串转整数
float_num = float(int_num)           # 整数转浮点数
```
