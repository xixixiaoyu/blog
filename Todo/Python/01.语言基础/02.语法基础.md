# Python 语法基础

## 最佳提示词

```
请详细介绍 Python 的基础语法，包括以下内容：

1. Python 语法特点（简洁性、可读性、缩进规则）
2. 变量命名规则与约定
3. 注释的使用方法（单行注释、多行注释、文档字符串）
4. 基本输入输出操作（print()、input() 函数）
5. 代码缩进规则与最佳实践
6. 语句分隔与续行
7. Python 标识符与关键字
8. 代码风格规范（PEP 8 基础）
9. 常见语法错误与调试技巧

请提供丰富的代码示例，每个语法点都应有对应的实际代码演示，并解释代码的执行逻辑。适合零基础学习者理解 Python 的基本语法结构。
```

## 学习要点

- 理解 Python 语法的设计哲学和特点
- 掌握变量的命名规则和良好命名习惯
- 学会使用注释提高代码可读性
- 掌握基本的输入输出操作
- 理解并正确使用缩进，这是 Python 语法的重要特点
- 了解 Python 的关键字和标识符规则
- 初步了解 PEP 8 代码风格规范
- 能够识别和解决常见的语法错误

## 实践练习

1. 创建一个 Python 文件，练习变量命名和赋值
2. 编写包含单行注释和多行注释的代码
3. 使用 print() 函数输出不同类型的数据
4. 使用 input() 函数获取用户输入并进行处理
5. 故意制造一些语法错误，学习如何阅读错误信息
6. 尝试使用不同的缩进方式，观察 Python 的反应

## 代码示例模板

```python
# 这是一个单行注释

"""
这是一个多行注释
可以跨越多行
通常用于文档说明
"""

def hello_world():
    """这是一个函数的文档字符串"""
    name = input("请输入你的名字: ")  # 获取用户输入
    print(f"Hello, {name}!")  # 格式化输出

if __name__ == "__main__":
    hello_world()
```
