## 学习要点

- 掌握条件语句的各种形式和使用场景
- 理解 for 循环和 while 循环的区别和适用情况
- 学会使用循环控制语句改变程序执行流程
- 掌握异常处理机制，编写健壮的代码
- 理解迭代器和生成器的工作原理
- 学会使用上下文管理器管理资源
- 了解控制结构的性能特点和优化技巧

## 代码示例模板

```python
# 条件语句示例
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "D"

# 三元运算符
result = "及格" if score >= 60 else "不及格"

# for 循环示例
for i in range(1, 6):
    print(f"第 {i} 次循环")

# while 循环示例
count = 0
while count < 5:
    print(f"计数: {count}")
    count += 1

# 异常处理示例
try:
    num = int(input("请输入一个数字: "))
    result = 10 / num
except ValueError:
    print("输入的不是有效数字")
except ZeroDivisionError:
    print("不能除以零")
else:
    print(f"结果是: {result}")
finally:
    print("程序执行完毕")

# 生成器示例
def fibonacci_generator(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# 上下文管理器示例
with open("example.txt", "w") as file:
    file.write("Hello, Python!")
```
