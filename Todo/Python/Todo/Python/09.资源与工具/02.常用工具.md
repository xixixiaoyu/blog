# Python 常用工具

## 最佳提示词

```
请详细介绍 Python 开发中的常用工具，包括以下内容：

1. 代码质量工具：
   - Black 代码格式化
   - isort 导入排序
   - flake8 代码检查
   - mypy 类型检查

2. 调试工具：
   - pdb 调试器
   - ipdb 增强调试器
   - pdb++ 调试器
   - 远程调试

3. 性能分析：
   - cProfile 性能分析
   - line_profiler 行级分析
   - memory_profiler 内存分析
   - py-spy 性能监控

4. 包管理工具：
   - pip 基础使用
   - pipenv 环境管理
   - poetry 依赖管理
   - conda 科学计算环境

5. 文档工具：
   - Sphinx 文档生成
   - MkDocs 文档系统
   - pdoc 自动文档
   - 文档字符串规范

请为每个工具提供详细的配置和使用方法，包括实际应用场景和最佳实践。
```

## 学习要点

- 掌握代码质量工具的使用
- 学会使用调试器进行问题排查
- 理解性能分析的方法
- 掌握包管理工具的选择
- 学会文档生成工具的使用

## 实践练习

1. 配置代码格式化和检查工具
2. 使用调试器排查代码问题
3. 进行性能分析和优化
4. 设置项目依赖管理
5. 生成项目文档

## 代码示例模板

```bash
# 代码质量工具配置
# pyproject.toml
[tool.black]
line-length = 88
target-version = ['py39']

[tool.isort]
profile = "black"
multi_line_output = 3

[tool.mypy]
python_version = "3.9"
warn_return_any = true
warn_unused_configs = true
disallow_untyped_defs = true

# 性能分析示例
python -m cProfile -o profile.stats my_script.py
python -m line_profiler my_script.py
python -m memory_profiler my_script.py

# 文档生成
sphinx-quickstart docs
mkdocs new my-project
pdoc --html my_module
```
