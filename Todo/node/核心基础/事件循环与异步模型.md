# 事件循环与异步模型

目标
- 理解 Node 的事件循环阶段、microtask 与 nextTick 的执行顺序与差异。

示例：执行顺序
```js
setImmediate(() => console.log('setImmediate'));
setTimeout(() => console.log('setTimeout 0'), 0);
Promise.resolve().then(() => console.log('promise microtask'));
process.nextTick(() => console.log('nextTick'));

// 典型输出：
// nextTick
// promise microtask
// setTimeout 0
// setImmediate
```

要点
- `process.nextTick` 比 microtask 更优先，谨慎使用以避免事件循环饥饿。
- I/O 回调在不同阶段执行，理解阶段有助于调优性能与时序。
