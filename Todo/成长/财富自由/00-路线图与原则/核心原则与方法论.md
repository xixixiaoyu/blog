## 第一部分：五项核心原则

### 1. 支付自己（Pay Yourself First）

收入到账后，**第一时间**将目标储蓄率（建议 ≥ 20%）转入独立投资账户，剩余部分才用于消费。这不是预算技巧，而是**优先级声明**——将未来的你视为最优先债权人。

这是建立自动化投资系统的第一步。

注意将「支付自己」用于偿还信用卡分期 —— 这只是在修复过去的错误，非构建未来。



### 2. 冗余与安全边际（Redundancy & Margin of Safety）

任何财务决策都需预留**可承受错误的空间**。安全边际是资产内在价值与支付价格的差距；冗余是系统缓冲（现金、保险、多元收入）。

- 评估一项投资：要求至少 30% 折扣于保守估算的内在价值
- 家庭财务：现金覆盖 6-12 个月必需支出
- 职业选择：技能组合需有「Plan B」行业可迁移

过度冗余。持有 30% 以上现金且长期不投入，是对抗通胀的慢性亏损。



### 3. 低成本与分散（Low Cost & Diversification）

成本是**确定的负收益**，分散是**免费的午餐**。用低于 0.5% 的总费率获取市场平均回报，通过跨资产、跨地域、跨策略分散，降低非系统性风险。

- 核心持仓（≥ 60% 资产）应配置于宽基指数（沪深 300、标普 500）
- 避免单一行业/个股仓位 > 10%
- 优先选择 A 类份额基金或 ETF，拒绝前端收费



### 4. 长期主义（Long-Termism）

将财富积累视为 **10 年以上的复利游戏** ，而非 1-3 年的投机。长期主义是**认知框架**：接受短期不可预测，专注长期大概率正确的事。

- 权益类资产持有期 ≥ 7 年（跨越一个完整库存周期）
- 对年化 8-12% 的回报感到满意，拒绝「翻倍」诱惑
- 定期投资（定投）跨越市场周期

长期主义≠永不卖出，需定期审查逻辑。避免市场下跌时恐慌离场，上涨时追高加仓。



### 5. 反脆弱（Antifragility）

系统不仅能抵御冲击，还能从波动中受益。财务反脆弱性 = **现金流多元 + 期权性收益 + 心理韧性**。

- 配置 5-10% 于「非对称机会」（如深度虚值期权、早期股权、危机债券）
- 主业外建立 1-2 个收入流（咨询、创作、投资分红）
- 保持 20% 以上时间/资本的「实验预算」，允许试错

误区：

- 将投机误认为反脆弱。买彩票是脆弱，买入恐慌后的优质资产才是反脆弱。
- 过度追求「受益」而忽视「抵御」。反脆弱的前提是**生存优先**。
- All-in 创业且无家庭现金储备，波动不是营养，是毁灭。



## 第二部分：五项执行方法

### 方法 1：OKR/KPI 资本管理

**目标**：将财务目标从模糊愿望转化为可衡量、可追踪的季度/年度结果。

1. **设定年度 O（Objective）**：如「2025 年净储蓄率达到 35%」
2. 拆解 KR（Key Results）
   - KR1：月度自动转账储蓄 15,000 元（责任人：自己，截止日期：每月 1 日）
   - KR2：将投资组合费率从 0.8% 降至 0.4%（责任人：自己，截止日期：Q2）
   - KR3：副业收入月均达 5,000 元（责任人：自己，截止日期：Q4）
3. 定义 KPI 仪表盘
   - 储蓄率、投资回报率、成本率、现金月数
4. **季度复盘**：评估完成度，调整下一季度 KR

**地清单**

- **谁**：你（主责任人）+ 伴侣（如适用，共识方）
- **何时**：每年 12 月设年度 OKR，每季度末月 25 日复盘
- 做什么
  - 更新个人资产负债表（模板见附录）
  - 对比 KPI 与目标值，偏差 > 20% 需写根因分析
  - 调整下季度自动转账金额或投资组合



### 方法 2：OODA 循环（Observe-Orient-Decide-Act）

**目标**：在重大财务决策（如购房、换工作、大额投资）中快速迭代认知，避免分析瘫痪。

1. **Observe（观察）**：收集原始数据。例：观察目标城市二手房成交量价、利率走势、个人现金流变化（耗时 1-2 周）
2. **Orient（判断）**：用心智模型解读数据。例：用「安全边际」判断房价是否高于租金折现值 30%（耗时 3-5 天）
3. **Decide（决策）**：生成备选方案。例：方案 A（立即买）、B（续租观望）、C（购小户型）（耗时 1 天）
4. **Act（行动）**：执行最小可行行动。例：方案 B→先与房东谈续租条款，锁定灵活性（耗时 1 周）
5. **循环**：每 2 周重复一次，直到决策信号清晰

**落地清单**

- **谁**：你 + 1-2 位「外部智囊」（不涉利益的朋友或付费顾问）
- **何时**：面对 > 50 万元或影响 > 3 年的决策时启动
- 做什么
  - 建立专属决策笔记本（Notion/Obsidian）
  - 每个循环记录「观察清单」与「判断依据」
  - 行动后设定「决策截止日」，逾期则执行默认选项（如不决策=不行动）



### 方法 3：复盘与事后分析（After-Action Review）

**目标**：从成功与失败中提取**可复用的规则**，而非沉溺于情绪。

1. **事件触发**：单笔投资亏损 > 15% 或盈利 > 30% 时启动
2. **记录事实**：买入时间、价格、逻辑、持仓期间市场变化（客观数据）
3. **对比预期**：当初预期是什么？实际差异点在哪？
4. **提炼洞察**：哪个假设错了？哪个工具好用？（聚焦**可改变**的因素）
5. **更新清单**：将洞察写入「个人投资负面清单」或「检查清单」

**落地清单**

- **谁**：你（主笔）+ 伴侣（旁听，提供视角）
- **何时**：触发事件后 7 天内完成，避免记忆失真
- 做什么
  - 使用模板（见下）书面化，非头脑复盘
  - 每月末回顾本月所有复盘，寻找「重复性错误」
  - 每季度将高频错误转化为「预承诺机制」的触发条件

```ts
// 复盘模板（极简版）
interface AfterActionReview {
  asset: string // 资产名称
  trigger: 'loss>15%' | 'gain>30%' | 'annual'
  facts: {
    entryDate: string
    entryLogic: string // 当初为什么买
    exitDate?: string
  }
  expectation: string // 预期发生什么
  reality: string // 实际发生什么
  insight: string // 核心洞察（一句话）
  ruleUpdate: string // 要新增或修改哪条规则
}
```



### 方法 4：清单化（Checklist Manifesto）

**目标**：将复杂决策转化为**防呆步骤**，规避认知偏误与情绪干扰。

1. **识别高频决策**：如「是否买入一只新基金」「是否接受新工作 Offer」
2. **提取关键检查项**：每个决策点提炼 3-5 个「否决项」和「加分项」
3. **强制书面化**：打印或保存在手机快捷指令中，决策前必须逐条勾选
4. **定期更新**：每季度根据复盘结果增删条目

**落地清单**

- **谁**：你（制定与执行）
- **何时**：每次触发对应决策时
- 做什么
  - **否决项**（任一答「否」即停止）：如「该基金经理任期 < 3 年？」「费率 > 0.8%？」
  - **加分项**（累计 < 3 分需警惕）：如「有 3 年以上实盘业绩记录 +1」「熊市跌幅小于指数 +1」
  - 决策后记录「清单外因素」，评估是否需新增检查项



### 方法 5：预承诺机制（Pre-Commitment）

**目标**：在理性时**锁定**未来行为，防止情绪期偏离轨道。

1. **识别脆弱时刻**：如「市场单日暴跌 5% 时想割肉」「连续上涨后想加杠杆」
2. **设计约束规则**：如「若沪深 300 单日跌幅 > 4%，自动买入 1 万元（提前设限价单）」
3. **引入外部监督**：告知伴侣/朋友你的规则，请其监督执行
4. **设置「熔断」**：连续 2 次触发预承诺后，强制冷静期 72 小时

**落地清单**

- **谁**：你（承诺人）+ 伴侣/挚友（见证人）
- **何时**：在年度 IPS 制定时同步设定
- 做什么
  - 书面签署《预承诺契约》（模板见附录）
  - 在交易软件中设置条件单，自动化执行
  - 每月例会检查「预承诺触发次数」，评估情绪稳定性



## 第三部分：决策框架与工具

### 框架 1：资产配置金字塔

```
        顶层（5-10%）：实验性资产（期权、早期股权、加密货币）
        -------------------------------------------------
        中层（20-30%）：卫星资产（行业主题、主动基金、REITs）
        -------------------------------------------------
        基石层（60-70%）：核心资产（宽基指数、国债、现金）
        -------------------------------------------------
        地基：保险与应急现金（6-12 个月支出）
```

金字塔的**重量**（比例）决定风险水平，**高度**（层级）决定反脆弱性。基石层不可动用，顶层可全部亏损而不影响生活。



### 框架 2：杠铃策略（Barbell Strategy）

**定义**：将 80-90% 资产置于**极端保守**（现金、国债），10-20% 置于**极端进攻**（非对称机会），**拒绝中间地带**（如平衡型基金）。

**适用场景**

- 风险厌恶型投资者
- 市场估值处于历史高位（如股债比 > 90% 分位）
- 职业收入波动大（如创业者）

**执行要点**

- 保守端：TIPS（通胀保值债）+ 短期国债，目标**保值**
- 进攻端：危机后的可转债、深度虚值看跌期权买方、行业黑天鹅后的龙头
- **纪律**：进攻端单笔亏损 > 50% 必须止损，防止侵蚀保守端



### 框架 3：Kelly 近似与风险预算

Kelly 准则告诉你：当胜率与赔率已知时，**最优下注比例**是多少。简化公式：

```
建议仓位 ≈ (胜率 × 赔率 - 败率) / 赔率
```

**实用版（不求推导）**

1. 风险预算法：将「可接受最大回撤」转化为「权益仓位上限」
   - 若可接受最大回撤 = 20%，历史熊市权益跌幅 ≈ 50%
   - 则权益仓位上限 ≈ 20% / 50% = 40%
2. Kelly 分数：将计算出的理论仓位 × 0.3-0.5（保守系数），防止过度自信
   - 理论仓位 60% × 0.5 = 实际仓位 30%

**适用边界**

- 仅适用于**重复性、概率可估**的决策（如定投宽基指数），不适用于一次性大额投机
- 胜率与赔率估计偏差 > 20% 时，公式失效，回归风险预算法



### 框架 4：再平衡纪律

**再平衡阈值设定**

- 带宽：资产类别偏离目标比例 ± 10% 绝对值（非相对值）
  - 例：权益目标 50%，当 > 60% 或 < 40% 时触发
- **频率**：**季度检查**，年度强制再平衡（避免税务/手续费损耗）
- **例外**：市场单日暴跌 > 8% 时，可临时触发再平衡（逆向操作）

**执行 SOP**

- 偏离 > 20% 且持续 > 30 天：必须再平衡

- 偏离 10-20%：可再平衡，但需记录理由
- 偏离 < 10%：不操作

卖出超配资产，买入低配资产，**优先在税收优惠账户内操作**
