# AI Prompt: Fiber 框架快速入门教程

请你扮演一位对性能有极致追求的 Go 语言导师，为初学者撰写一篇 Fiber 框架的“快速开始”教程。

**文章目标：**
旨在引导读者在最短的时间内上手 Fiber，完成从安装、编写并运行第一个 “Hello, World!” 应用的完整流程。文章需要突出 Fiber 的简洁性和高性能特性，让读者直观感受到它的魅力。

**核心内容：**

1.  **前置准备**
    *   明确指出读者需要预先安装好 Go 语言环境（建议 1.18+）。
    *   指导读者创建一个新的项目目录，并使用 `go mod init <module-name>` 初始化 Go 模块。

2.  **安装 Fiber**
    *   提供清晰的 `go get` 命令来安装 Fiber v2。
    *   命令：`go get -u github.com/gofiber/fiber/v2`
    *   简单解释该命令的作用：下载并安装 Fiber v2 版本的包及其依赖。

3.  **编写你的第一个 Fiber 应用：“Hello, World!”**
    *   **创建 `main.go` 文件**：指导读者在项目根目录下创建此文件。
    *   **提供完整的代码**：
        ```go
        package main

        import (
            "log"
            "github.com/gofiber/fiber/v2"
        )

        func main() {
            // 1. 创建一个新的 Fiber 实例
            app := fiber.New()

            // 2. 定义一个 GET 路由
            app.Get("/", func(c *fiber.Ctx) error {
                // 3. 发送一个字符串响应
                return c.SendString("Hello, World 👋!")
            })

            // 4. 启动服务器并监听在 3000 端口
            log.Fatal(app.Listen(":3000"))
        }
        ```
    *   **逐行代码讲解**：
        1.  `fiber.New()`：解释它创建了一个新的 Fiber 应用实例。
        2.  `app.Get("/", ...)`：解释这行代码注册了一个 HTTP GET 方法的路由，路径是 `/`。
        3.  `func(c *fiber.Ctx) error`：介绍 `fiber.Ctx` 是 Fiber 的上下文对象，封装了请求和响应。注意处理函数需要返回一个 `error`。
        4.  `c.SendString(...)`：解释这是 Fiber 中发送字符串响应的便捷方法。
        5.  `app.Listen(":3000")`：解释它启动了 HTTP 服务器，并处理了可能的启动错误。

4.  **运行与验证**
    *   指导读者在终端中运行 `go run main.go`。
    *   告诉读者服务器成功启动后会看到的 Fiber 启动艺术字和监听地址。
    *   引导读者打开浏览器或使用 `curl` 命令访问 `http://localhost:3000`。
    *   展示预期的响应结果 `Hello, World 👋!`，给读者即时的正反馈。

5.  **与 `net/http` 的兼容性（可选，但建议提及）**
    *   简要提及，虽然 Fiber 基于 `fasthttp`，但它也提供了与 `net/http` 生态交互的桥梁，例如通过 `fiber.Handler` 将 `net/http` 的处理器转换为 Fiber 处理器。这可以打消一些开发者对生态兼容性的顾虑。

**总结：**
祝贺读者成功地用这个号称“最快之一”的 Go 框架构建了他们的第一个应用。鼓励他们去感受 Fiber 带来的开发速度和运行速度的双重提升，并可以继续探索 Fiber 丰富的中间件和高级功能。
